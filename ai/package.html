Artificial Intelligence

This package contains a framework for experimenting with different artificial intelligence approaches. Subpackages contain implementations of each approach.

The framework assumes a neural network structure that is built incrementally based on feedback during the learning experience (see Cziko's book for perceptual control feedback). Each neuron receives sensory input, which is modifies according to an adaptive function, and finally an action is output. A single neuron can interact directly with its environment, or multi-layer netowork of neurons can act in a delegated manner.

See the unit tests for this package for examples.
