The program above is a transliteration of an example task a
    classical
    human computer could follow to perform a precise
    computation, formalized and styled according
    to Alan
    Turing's method:



"You are given a long paper/ticker tape with boxes drawn on it,
and within each one you can read or write symbols.  All boxes start
with 0 (zero) written in them.  To perform a computation, you maintain
a state of interaction with the tape in your head, starting with
the start state and then follow a "program" of instructions
that exactly specifies the calculation.  Like this:

In state start, if the symbol seen is 0, write a 1; now if the
symbol seen is 1, shift to the right, and change into
state daisy; in state daisy, if the symbol seen is 0,
write a 1 and change to state cow; if the symbol seen is a 1,
change to state end; in state cow, write a 1 and change
to state end; in state end stop"

    


    
While this may seem obtuse, Turing supposed that by breaking
    down computations, even complex ones, into the simplest of parts,
    it might be possible to build a machine out of these parts, that
    could automatically run the program of instructions to carry out
    the computation.  He hypothesized a particular form for
    this "automatic
    machine", with a ticker tape and a mechanical processor
    interacting with it, as described above.  This idealized computer
    turned out to be able to represet a stunning variety of
    computational and algorithmic processes, and so became the model
    for successive attempts to actually build one, which famously
    succeeded.


    
As the descendent computers and computer systems have
    developed, a wide variety of "languages" have also been developed
    to tranliterate program specifications created by humans into
    symbolic/material forms that can be stored and processed directly
    by computers.


    
After many generations and manifestations, a langauge known
    as JavaScript
    (named for a type of programming called "scripting", and for an
    influential predecessor
    language Java),
    and an
    associated JavaScript
    Object Notation (JSON), have emerged as popular programming
    and data description languages (respectively) for the
    inter-networked computing and communication system known as the
    InterWebz.


    
And so, in celebration of Alan Turing's 100th birthday and in
    recognition of one of the greatest outcomes of his work, here is a
    JSON programming interface to a Turing Machine written in
    JavaScript (see the linked turing.js).


    
The program above is a placeholder to get you started, but you
    can edit the program as you want to manipulate the tape as you
    want and, in principle, construct a program that is equivalent to
    any software running in the world if the tape is long enough, and
    your patience boundless.
